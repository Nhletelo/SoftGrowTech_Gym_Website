# 🏋️ Single Page Gym Website

A modern single-page gym website built using **HTML5, CSS3, and JavaScript**.  
The website showcases gym facilities, services, and membership plans in a clean and scrollable layout similar to a commercial fitness website.

---

## 🚀 Features

- Responsive single-page layout
- Modern hero section with call-to-action
- Gym facilities section
- Services offered by the gym
- Membership pricing plans
- Contact form with JavaScript interaction
- Smooth scrolling navigation
- Hover animations and modern UI design

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
